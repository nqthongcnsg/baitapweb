Convert files to excel

Lấy dữ liêu và đổ dữ liệu lên table , chuyển dữ liệu đó thành excel
Sử dụng ajax của jquery-3.6.0  để lấy dữ liệu
Dữ liệu ở đây là file shop.sql
Lấy dữ liệu thông qua file data.php 

Sau khi lấy dữ từ data.php nhờ vào ajax thì show lên table

Để chuyển thành file excel thì click vào button "Chuyển thành file excel"
